jQuery Resizable

About the Experiment

This experiment demonstrates the Resizable widget in jQuery in which you can change the size of an element by dragging with a mouse. Use the mouse to extend or contract the element.


Code Snippet:

            
$(document).ready(function () {
    $("#resizable").resizable({
        alsoResize: "#also",
        aspectRatio: true,
        autoHide: true,
        ghost: true,
        maxHeight: 300
    });
});

Description

#resizable is the id for the div that needs to be resized. Another div with id "#also" is created. .resizable is used to create a resizable element. We can specify several options such as maxHeight, maxWidth, minHeight, minWidth, autoHide, ghost and so on. Here, I have experimented with maxHeight, ghost, autoHide and alsoResize.
  • alsoResize is used to specify another element that is to be synchronously resized when the current element is resized. Since I have specified alsoResize: "#also", the div with id "also" is resized simultaneously.
  • aspectRatio is used to retain the same aspectRatio. I have set aspectRatio to be true. You can also specify a number which forces the element to maintain a specific aspect ratio during resizing.
  • autoHide is used to hide the resize handle unless the mouse is hovered over it.
  • ghost: If it is set to true, a semi-transparent helper element is shown for resizing.
  • maxHeight is used to specify the maximum height to which the element can be resized.
