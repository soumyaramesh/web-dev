AngularJS Forms

About the Experiment

In tis experiment, I have demonstrated a sample form using AngularJS.
An AngularJS form is a collection of input controls for the purpose of grouping related controls together. Form and controls provide validation services for the user to be notified of invalid input before submitting a form. This provides a better user experience than server-side validation alone because the user gets instant feedback on how to correct the error.

Code Snippet

angularApp Script 
            var app = angular.module('myApp', []);
            app.controller('FormController', function ($scope) {
            $scope.content = {firstName:"",lastName:"",email:""};
            $scope.reset = function () {
            $scope.person = angular.copy($scope.content);
            };
            $scope.reset();
            });
html file <div ng-app="myApp" ng-controller="FormController"> <form> First Name:<br> <input type="text" ng-model="person.firstName"><br> Last Name:<br> <input type="text" ng-model="person.lastName"><br /> Email:<br> <input type="email" ng-model="person.email"><br /> <br><br> <button ng-click="reset()">RESET</button> </form> <div class="container-fluid" style="background-color:antiquewhite"> <h4>Form JSON:</h4> {{person}} <h4> Form Content: </h4> First Name: {{person.firstName}}<br /> Last Name: {{person.lastName}}<br /> Email: {{person.email}}<br /> </div> </div>
Description

ng-app directive is used to define the AngularJS application. The ng-controller directive defines the application controller which is the FormController in this example. The ng-model directive binds the input elements to the person object in the model. The FormController controller initializes the values for the person object and defines the reset() method. The reset() method sets the person object values equal to the content object values. The ng-click directive invokes the reset() method when the reset button is clicked.
