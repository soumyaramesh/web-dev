About the Experiment

This week is similar to last week's experiments, except that instead of using static data, I have used MongoDB to fetch and insert data.
In this experiment, I have inserted some book information into the database and retrieved them and displayed it in a table.

Below, I have displayed the contents of the database using RockMongo

Code Snippet:

server.js

var mongoose = require('mongoose');

var BookSchema = new mongoose.Schema({
    title: String,
    author: String,
    chapters: [ChapterSchema]
}, { collection: "book" });
//Collection
var Book = mongoose.model("book", BookSchema);


exp1.html

<table class="table-bordered table-condensed table-striped">
    <tbody>
        <tr ng-repeat="b in books">
                <td>b.title</td>
                <td>b.author</td>
        </tr>
    </tbody>
</table>

Description :


require is similar to import in Java. It is used to import a library. Here, I am using the mongoose library to use the Mongoose Framework which helps us interact with MongoDB.
I have created a new BookSchema with title, author and chapters where chapters is an array of the type ChapterSchema which we will see in future experiments.
mongoose.model is used to create a collection called book and associate it to the BookSchema. I have then displayed the contents of the database as a table.
