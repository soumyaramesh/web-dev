About the Experiment

This experiment is an extension of the previous experiment where I have created a new button to delete a book from the database.

The figure below shows the contents of the database after a delete is performed on the book with title "Girl on the train".


Code Snippet:

        
        
server.js
var bodyParser = require('body-parser');
app.use(bodyParser.json()); // for parsing application/json

app.delete("/book/:id", function (req, res) {
    Book.findById(req.params.id, function (err, data) {
        data.remove();
        Book.find(function (err, data) {
            res.json(data)
        });
    });
});

clientApp.js
            $scope.remove = function (id) {
        $http.delete("/book/" + id)
        .success(function (response) {
            $scope.books = response;
        });
    };

Description

When the X delete button is clicked, the remove handler is called which used to generate a delete using $http.delete which looks for a match in the form of /book/id.
If it is a success, Book.findById is used to find the book that is to be deleted and data.remove() deletes that record.

In package.json, we include express,mongoose and body-parser as a dependencies.
