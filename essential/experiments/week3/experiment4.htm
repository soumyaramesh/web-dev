Fading In and Out

About the Experiment

This is my first jQuery experiment. Hover over the blue div to observe the fade effects when your mouse enters and leaves the div. Click the orange div and it will fade out.



Hover Over Me!


Click Me!

Code Snippet:

$(document).ready(function () {
    $("#blue").mouseenter(function () {
        $("#blue").fadeTo('fast', 1)
    });
    $("#blue").mouseleave(function () {
        $("#blue").fadeTo('fast', 0.5)
    });
    $("#orange").click(function () {
        $("#orange").fadeOut('slow');
    });
});

$( ) marks the start of a jQuery object. $(document) is a jQuery object for the HTML document and .ready means that the function action take place when the document is loaded and ready. $("#blue") and $("#orange") are used to access the corresponding div elements. .fadeTo is the function applied to that element. .fadeTo function takes two parameters: one to indicate the speed of fade and the other to indicate the opacity.
