Index of First Vowel in the string

About the Experiment

This experiment uses a form to obtain a user input string and on submit, executes a javascript script to produce the index of the first vowel in the string and the vowel. It is assumed that the string starts with an index of 0.




Code Snippet:

function isvowel(elem)
         {
            var vowels="aeiouAEIOU";
            var str=elem.value;
            var chr=" ";
            var nIndx=-1;
            var pos=0;
            for(; pos<str.length;pos++) {
               chr=str.charAt(pos);
               if(vowels.indexof(chr)!=-1) {
                  nindx=vowels.indexOf(chr);
                  break;
                  }
             }
           }

The variable vowels is initialized to contain a string of all vowels. The user input string is looped through one character at a time and the .indexof(chr) function is used to check if that character is present in the vowel string. If it is, the current position and character is returned. Else, there is no vowel in the string.
