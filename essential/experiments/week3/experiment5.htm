To Do List Using jQuery

About the Experiment

This is a To-Do list. An item can be added by entering the item in the text box and clicking Add. An item can be removed by clicking on that item.


Add


Code Snippet:

$(document).ready(function () {
    $("#button").click(function () {
        var li = $('input[name=listItem]').val();
        $(".list").append('<div class="delItem">' + li + '</div>');
    });
    
    $(document).on('click', '.delItem', function () {
        $(this).remove()
    });
});

When the Add button is clicked, I have declared the variable li to obtain the value of the input obtained by the user. On each addition of an item, I modify the HTML div element with class="listItem" by appending a each item in a new div with class="delItem" dynamically. I use this class delItem to delete an item when it is clicked using $(this).remove.
