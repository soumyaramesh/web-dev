About the Experiment

I have demonstrated a few new HTML5 form input types such as color,url,number,date and range

The <color> input type

The color input type allows for input fields to pick a color. A color pop-up is shown depending on browser support. Note: Internet Explorer does not support color

Pick a color of your choice:
Code Snippet: 
            <form>
            <input type="color" name="mycolor">
            </form>

The <url> input type

The url input type is used for an input that is supposed to contain a url. The url field is validated before submit depending on browser support.

Add a link to your blog:

Code Snippet: 

            <form>
            <input type="url" name="myurl">
            </form>

The <number> input type

The number input type should be used when the input it supposed to contain a numeric value. We can add restrictions like max, min, maxlength,required,size etc

Enter your age (between 18 and 60):

Code Snippet: 

            <form>
            <input type="number" name="mynum" min="18" max="60" value="18">
            </form>

The <date> element

The date input type is used when the input firld is required to contain a date. Restrictions such as min and max can be specified.

Enter you date of joining:

Code Snippet: 

            <form>
            <input type="date" name="joinday" max="2015-01-31">
            </form>

The <range> input type

The range input type is used when the input value should fall within a range.

Code Snippet: 

  
            <form>
            <input type="range" min = "0" max="100">
            <form>
