About the Experiment

I have demonstrated some of the new media elements in HTML5 such as audio, video and youtube video embedding in a web page.

HTML5 Audio and Video

Audio and Video are notable new features of HTML5. Previously, multimedia files could only be played using a plug-in like flash. I have included a sample audio clip below: (Courtesy: US Fish and Wildlife Service)

Code Snippet: 

  
            <audio controls>
            <source src="audio/test_audio_clip.mp3" type="audio/mpeg">
            Your browser does not support the audio element.
            </audio>

Similarly it is very simple to embed a video using HTML5

Code Snippet: 

  
            <video width="300" height="240">
            <source src="video/test_movie.mp4" type="video/mp4">
            Your browser does not support the video element.
            </video>

Embedding a Youtube Video

We could also embed a youtube video in a web page using iframe,object or embed. I have embedded a youtube video of CS5610:Intro to HTML by Prof Jose Annunziato below using iframe:

Code Snippet: 

  
            <iframe width="420" height="350"
            src="//www.youtube.com/embed/rvFdP8taVQo">
            </iframe>
