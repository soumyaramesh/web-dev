About the Experiment

I have demonstrated a few new HTML5 form elements such as datalist, keygen and output.

The <datalist> form element

The datalist element is used to provide a drop-down list of pre-defined list of options for the user to input.

Pick a color or type your favourite color:

Code Snippet: 

            <form>
            <input list="colors" name="colors">
            <datalist id="colors">
            <option value="red">
            <option value="green">
            </datalist>
            </form>
            </form>

The <keygen> tag

The keygen tag is used to generate generates a public/private key pair and then creates a certificate request. This certificate request will be sent to a Certificate Authority (CA). The CA creates a certificate and sends it back to the browser, therefore using it for client authentication.

Enter your Username:


Encryption:


Code Snippet: 

            <form>
            Enter your username
            <input type="text" name="username">
            <Encryption>
            <keygen name="keygen">
            </form>

The <output> element

The output element is used to show the output of a calculation.

+ =

Code Snippet: 

            <form oninput="c.value = parseInt(a.value) + parseInt(b.value)">
            <input name="a" type="number"> +
            <input name="b" type="number"> =
            <output name="c"></output>
            </form>
