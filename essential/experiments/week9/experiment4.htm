Search Bar


About the Experiment

In this experiment, I created a search bar which is used to search the database based on the user's input and display posts which match that particular search query.

Code Snippet:

home.html <input class="form-control" id="searchbox" placeholder="Search" ng-model="searchbox" /> <br /> <div ng-repeat="post in posts | orderBy:'-upvotes' |filter:searchbox">

Description :


A new search bar is created which is a form control of input type where the user can enter the search query. This query is binded using ng-model = "searchbox"
I have added a search filter using filter:searchbox which filters the posts based on the search query.
