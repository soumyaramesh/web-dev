About the Experiment

Node.js is an open source, cross-platform runtime environment for server-side and networking applications. Node.js applications are written in JavaScript, and can be run within the Node.js runtime.
In this experiment, I setup NodeJS and demonstrated a simple NodeJS application which displays information from a static web page which is stored in a common public directory.

Code Snippet:

var express = require('express'); 
var app = express ; 
  
app.use(express.static(__dirname + '/public'));

hello.html
<h1> Hello! from static hello.html</h1>

Description :


require is similar to import in Java. It is used to import a library. Here, I am using the express library to use the Express Framework.
app.use(express.static(__dirname + '/public')) is used to represent that the static content is present in my current directory appended with public.
hello.html is my static page whose content is displayed when the url is matched with "hello.html".
