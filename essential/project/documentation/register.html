Register and Login


Register Page


The Register Page contains a form for a new user to sign up. The user can sign up with a username and password. On successful registration, a new user object is created and stored in the database, after which the page will be redirected to the Profile Page. Here's a screenshot of the Registration page:



Login Page


The Login Page contains a form for a new user to enter his/her username and password and login. The usernames and passwords are stored in the UserModel which is retrieved at the time of login and authenticated using the LocalStrategy. PassportJS is used for this purpose.
