Add a new Post



The Add Post page contains a form for an authenticated logged in user to add a new post. Again, this page is restricted to authenticated users. A user may add a direct link or text content. If the content is a direct link, the page is redirected to that link when the post is clicked. Otherwise, the post details page will be displayed.
A user can associate his/her post with a tag indicating the category that the post belongs to. The user can choose from the auto-complete tags in the drop-down or enter a new tag. If a new tag is entered, a tag object is created and saved in the Tag Model. The new post is saved in the database and dynamically added to the list of posts which are updated on the home page.
