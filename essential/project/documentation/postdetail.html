Post Detail Page



The Post Detail Page displays a post's detailed information such as the post title, content, author, tag and upvote count. Below this, there is a section for the list of comments for that post. A user can post a comment only when he is logged in. A logged in user can see a form to post a new comment and can upvote and downvote comments.
