PostIt!

Overview

PostIt is a website which serves as a Q/A, entertainment, social networking or news forum where registered users can submit posts, comments, share direct links and other content. PostIt allows users to upvote/downvote posts and comments, and tag a post with a particular category, similar to Reddit, StackOverflow or Quora. I have added an additional social networking feature by allowing a user to follow another user. Most information on PostIt is public, allowing users to consume the content of the website without having to login. I built this application as a part of my Web Development Graduate course at Northeastern University. The motivation behind this idea is primarily the amount of help and entertainment that I have recieved from silmilar forums.
To summarize, PostIt provides the following features:
  • User Registration
  • Login
  • Access Profile
  • View all Posts
  • Add a Post
  • Add a Comment
  • View Posts by Tag/Category
  • Upvote Post or Comment
  • Downvote Post or Comment
  • View another User's Profile
  • Follow a User
  • Unfollow a User
