User Detail Page



The Public User Detail Page displays a user's detailed information such as the username, the user's post information including the post title, tag and upvote count and the user's following list.
When a user is logged in and views another user's page, apart from the above information, a Follow/Unfollow button is displayed based on whether the current user is following the other user or not. When the current user clicks the follow button, the other user is added to the current user's following list. If the unfollow button is clicked, that user is removed from the current user's folowing list.
