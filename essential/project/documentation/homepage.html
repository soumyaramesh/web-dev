Homepage


The Home Page consists of a responsive Navigation Bar with options to navigate to the Login, Register or Home views.
The home page displays all the post titles in order of their upvote count along with a link to the author, tag and comments of the post. The up and down arrow glyphicons are used to upvote and downvote a post. This function is available on when a user is logged in. Else, an alert is displayed. Here's a screenshot of the Home page:
